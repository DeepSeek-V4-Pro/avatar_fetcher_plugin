# 头像获取插件 (Avatar Fetcher Plugin)

通过指令获取 @用户 或指定 QQ 号的头像，以图片形式发送。

## 使用方法

```
/avatar @用户名     # 获取被 @用户 的头像
/avatar QQ号        # 获取指定 QQ 号的头像
获取头像 @用户名     # 同上
获取头像 QQ号        # 同上
```

## 配置说明

| 配置项 | 说明 | 默认值 |
|---|---|---|
| `plugin.enabled` | 插件总开关 | `false` |

## 依赖

- `aiohttp` >= 3.0.0

## 常见问题

**Q：提示"获取头像失败"？**
头像从 QQ 公开头像接口 `q1.qlogo.cn` 获取，请确认网络正常且 QQ 号有效。
